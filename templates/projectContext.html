Introduction

Camera (Wild) traps are widely used to monitor biodiversity and population density of animal species, by collecting large quantities of image data. There are several advantages on automating species classification from this data, such as less time spent in analyzing photos and an easier count of occurrences in specific locations. The goal of this project was to develop a deep learning model to correctly classify specific animal species based on camera trap images, based on the dataset provided by the iWildCam 2019 Kaggle Challenge. We used Convolutional Neural Networks (CNNs), with the Keras interface for TensorFlow Python library. CNNs are a specific type of artificial neural network, frequently used for image data processing and analysis
